ABOUT: We have reimagined Piet Mondrian’s Broadway Boogie Woogie as a traffic data visualization, reinvigorating the century-old abstract art tradition for a 21st century context.
The NYC Open Data web platform hosts many datasets related to mobility, the most intriguing of which is the DOT’s real-time traffic speed dataset. We humbly recognize that the DOT has already created a much more useful visualization of traffic speeds which can be found at the following location: NYC TMC. We’ve given ourselves a much simpler goal under the protections of artist expression to create a dynamic, two-dimensional update to Broadway Boogie Woogie. This dataset is updated every half hour, so the image will change in kind.

DATA SOURCE: NYC Open Data Portal API
